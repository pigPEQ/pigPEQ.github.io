使用Anki高效记忆

Anki是什么？

Anki是一款记忆闪卡应用，它所采用算法SuperMemory2是一种拟人记忆曲线的算法，能够针对用户的刷题记录进行降频复习，从而减少无必要的二次复盘。例如你刷到了一题你已经感觉基本掌握的试题，在你为该题选择“犹豫/想起”的标签时就决定该题的简易度，目前的最高简易度为130%，Anki初次以2天为间隔为你推送一次该试题，如果你觉得还是很ok，该题的推送间隔会被继续拉长，如4天，并降低该题的保留度。

为什么使用Anki

艾宾浩斯遗忘曲线

遗忘曲线显示了大脑对新信息的记忆是如何衰退的，保留率下降速度最快的地方是在20分钟后，曲线在一天后趋于平稳。学习曲线则与遗忘曲线相反，它说明了我们学习新信息的速度。

间隔效应

尝试将学习知识的过程分布在多个分散的时间区间里，我们反而能够更好地回想起这些知识。当我们使用间隔重复来学习时，遗忘曲线就会发生一定的变化。

如何用好Anki

Anki支持windows、Mac os、Android、IOS，用户可以将牌组资源上传到账号空间，并共享给来自全世界各地的Anki用户！Anki官方地址(https://apps.ankiweb.net/)，当然你也可以访问(Anki中国)。

如果你想深入了解Anki更多的用法，那么这一篇长达三万字的长文 量子物理学家是如何使用 Anki 的？你值得一阅，原文是 Augmenting Long-term Memory ，其作者 Michael Nielsen 是量子物理学家、科学作家、编程研究者，现供职于 Y Combinator Research 。他在《自然》等期刊发表过多篇高引论文，他写作的量子力学和深度学习教材也广受欢迎。作者在该文记录了用 Anki 增强科研和生活中长期记忆的经验，同时也写了他对广义上的记忆软件的深刻思考。

Anki卡片支持录入

  • 语音
  • 视频
  • 文本(填空、选择)
  • ORC
  • 图片

Anki的基本用法：选择题闪卡的批量制作

本教程的可行性不局限于windows(作者使用的是Mac os)，以下是教程需要的支持工具列表：

  • Microsoft Word
  • Microsoft Excel
  • 记事本(文本编辑)
  • Anki

制卡思路

思路很重要，在开始详细的教程之前，我们首先需要明确的是我们的整体制卡流程，一套科学严谨的制卡规范能够带来更高的制卡效率：

  1. 在Microsoft Word中规范化试题格式。
  2. 将文本转换表格后拷贝到Microsoft Excel。
  3. 单元格整合，符合卡片模板：问题（1列）+ 选项（1列）+ 答案（1列）+ 解析（1列）。
  4. 拷贝到UTF-8格式的文本文件。
  5. 将文本文件导入到Anki中。

如果你理解了如上所述的制卡流程，那么恭喜你基本已经掌握如何制做记忆卡片的核心了，至于为什么提了这么多，是因为下面的“试题格式规范化”、“单元格整合”步骤看起来很繁琐，各种公式会让你眼花缭乱，理解起来可能也很困难。

制卡流程

这里准备了一套2017年司法考试真题，为了方便展示，我截选了其中的前十道题，题干部分包含了问题、选项、答案和解析四个部分；本方法参考了(anki|批量制作选择题的又一种思路 - 黄河的六一的文章 - 知乎)，并在作者方法的基础上加以拓展和补充；本教程中选用的Anki模板：NetTeel：【Anki模板】Monokai风格之随机选项多功能模板1.0（单选、多选二合一）。

规范化试题格式

  • 清除无关数据信息，题目中仅需保留题干、答案及解析部分，其余信息一概删除，包括答案标签和解析标签，这两部分我会在规范化过程中利用换行符将其替换。
  • 清除所有空格、制表符(^t)和换行符(^p)
  • 用换行符替换所有题号(^#.)、选项号(^$.)、答案标签(“试题答案”)、解析标签(“试题解析”)
  • 规范格式试题样本。

文本转换表格

  1. 在Microsoft Word中，全选文字→菜单栏→插入→将文字转化为表格。
  2. 可以看到转化为了一个10行7列的表格，10行代表了10道题，7列分别代表题目、选项(ABCD)、答案、解析。
  3. 将表格全部复制到Excel中。

单元格整合

Excel中的视图：

如上图所示，在H1中填入公式=B1&CHAR(10)&C1&CHAR(10)&D1&CHAR(10)&E1将B1至E1合并，其中 CHAR(10) 的意思是自动换行，所以H1中的内容是不连续的。 H1合并完成后下拉，保证该列都是B至E的合并项。

在I列输入{{c1::____}}，该组合在Anki中时挖空的标签，在闪卡正面会显示一个下划线。在J1输入A1&I1，即问题加下划线的组合。

columns contents
J 记忆的卡片正面的问题部分
H 选项组合，由ABCD四个选项加换行符组成
F 试题答案
G 试题解析

结构上一目了然，将J、H、F、H列全部复制后以值的形式保存到新的四列，题目模板制作完成。

Excel → txt(UTF-8)

打开记事本或者文本编辑器，复制粘贴表格的内容，最终以UTF-8的纯文本格式保存到本地。

导入Anki

打开Anki，选择一个已存在的记忆库或者新建一个记忆库，导入UTF-8的文本文件。

注意以下情况会导致制卡失败或显示异常：

  • Excel表中拷贝到文本编辑器中的数据不是值的形式。
  • 最终表数据一定从A1开始，否则导入到文本编辑器中的样式不符合模板要求。

左侧为错误文本样式，右侧为正确文本样式。尽管左边的样式可能看起来更工整一些，但还是右侧的符合模板要求。

补充

很多情况下试题答案和题干是分开的，这种情况需要将题干与答案分开处理，将试题答案部分单独拷贝到一个文档中并对其进行格式规范化，其次将题干文字转化的表格在右侧插入一列，将规范化的试题答案部分拷贝在这一列中，后续步骤与上述一致。

自建Anki同步服务器

由于Anki官方的服务器在国外，所以账户上的卡组向手机同步数据时速度很慢，科学上网的效果也仍然不是很理想，所以自己动手在还未到期Aliyun服务器中搭建一个数据服务器来方便个人数据的同步。

服务器安装

ankisyncd服务器安装依赖：

  • python3
  • pip3
  • git

1.进入云主机，如果没安装相应版本，请先安装环境。
2.环境检查没问题后，从github上clone服务器源码：

3.安装git子模块和python模块环境

服务器配置

1.安装完成后，如果有需要可以更改服务器的配置文件ankisyncd.conf。

  • host对应服务器的监听地址，默认0.0.0.0表示接受所有IP请求，如果设置为127.0.0.1，那么将无法接受外网的请求；
  • port对应监听服务器的端口；
  • data_root对应同步数据的保存目录；
  • base_url对应基本的卡组文本数据目录；
  • base_media_url对应媒体文件目录；
  • base_media_url对应媒体文件目录。

2.添加服务器用户，并为其设定密码。

服务器运行

在anki-sync-server目录下后台启动服务器，即ankisyncd文件。

或者使用screen：

在网页中输入你的主机IP+端口号出现Anki Sync Server，说明服务器创建运行成功，如果没有显示，请检查。

Anki PC端配置

在Anki组件添加下打开插件安装目录addons21，在其目录下创建文件夹ankisyncd，在ankisyncd文件夹下创建文件_ ini _.py，然后重启anki后点击同步，输入之前添加的服务器用户及密码，添加卡组文件，完成pc端的文件上传。

注意，如果你需要更改端口号必须保证在安全组中放行了该端口。

Anki 安卓端配置(暂不支持ios)

在手机上打开anki，点击设置–高级设置–自定义同步服务器，填写相应的地址后，刷新同步按钮，输入服务器用户及密码，选择从AnkiWeb上传或下载卡组资源。
