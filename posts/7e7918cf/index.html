如何优雅地使用Mac

2020-07-24- 2021-01-05
2.2k- 9m
教程-工具Mac

  Mac最大的优势是GUI与命令行的完美结合，本篇教程指导你如何DIY个性化终端，打造舒适的终端体验环境。

ZSH

  Shell是Linux/Unix系统的外壳，Mac OS系统中也采用了shell，负责外界与内核之间的交互，将用户的命令转化为内核语言传递给内核，内核根据指令进行工作并将结果反馈于窗口上。简单的说就是通过一串简洁的指令执行一套逻辑可能很复杂的任务，从而提升人们的工作效率。目前常用的Linux系统和OS X中的shell默认采用了bash命令行，而在Mac OS中其实还预装了zsh命令行，这款shell非常的强大，被大众亲切地称为“终极Shell”，换上了可以说是鸟枪换炮了，它支持的功能比bash等丰富许多，感兴趣的朋友可以去了解一下。

  1. 你可以通过以下指令查询当前系统中装载了哪些shell，其实就是查看/etc目录下的shells文件：
cat etc/shells

  1. 知道了自己的系统中已装载了哪些shell后，我们就可以去查看当前使用的shell并去切换它：
echo $SHELLS  //输出环境变量SHELLS
chsh -s /bin/zsh   //切换zsh
  1. 输入用户的密码后shell就切换为zsh了，可以输入echo $SHELLS检查一下是否切换成功

iTerm2

 相比于Mac OS自带的终端，iTerm2对用户可是友好多了。iTerm2能够支持主题选择、声明高亮、自动填充建议、隐藏用户名和主机名、分屏效果等。

iTerm2的下载地址

配置主题

 iTerm2安装好后我们可以为其自定义一个主题，主题包可额外下载导入，这里安利一款好看的主题：Solarized

 开源项目地址

 如果你已经安装了git，可以直接使用以下命令安装：

$ git clone git://github.com/altercation/solarized.git

 打开preference，选择colors，点击右下角的Color Presets，从下拉框中选择一个主题或者import一个主题包,比如Solarized主题，打开解压的文件夹，选择其中的iterm2-colors-solarized文件夹中的文件。

配置背景图片

点击preference，选择windows，在bakcground-image栏下enable复选框选中后，从本地文件中选择一张图片作为背景，左侧的Transparency可以设置图片的透明度。

oh my zsh

oh my zsh是对zsh主题的拓展，不仅可以使你的终端界面更加炫酷，而且还支持更丰富的插件功能。

 开源项目地址

安装oh my zsh

# curl安装
sh -c "$(curl -fsSL https://raw.githubusercontent.com/robbyrussell/oh-myzsh/master/tools/install.sh)"

# wget安装
sh -c "$(wget https://raw.githubusercontent.com/robbyrussell/oh-my-zsh/master/tools/install.sh -O -)"

 如果见到下面这个界面时，说明安装成功。

 在确保当前shell已经切换为zsh的前提下，打开配置文件~/.zshrc

vim ~/.zshrc

#或者直接用记事本打开
open ~/.zshrc

#配置完成后生效配置文件
source ~/.zshrc

  • 修改主题ZSH_THEME="agnoster"，agnoster是最花哨的主题之一，主题库目录位于 ~/.oh-my-zsh/themes下。

    更多主题请参考https://github.com/ohmyzsh/ohmyzsh/wiki/Themes

  • 添加插件plugins=(xxx)，同时引入插件资源source xxx.zsh

    更多主题请参考https://github.com/ohmyzsh/ohmyzsh/wiki/Plugins

添加代码高亮

#.zshrc文件配置
plugins=(zsh-syntax-highlighting) 	#使用插件

#引入插件资源
source /Users/peggy/.oh-my-zsh/zsh-syntax-highlighting/zsh-syntax-highlighting.zsh

zsh-syntax-highlighting插件的开源地址

添加命令建议、补全和历史记忆

#.zshrc文件配置
plugins=(zsh-autosuggestions) 	#使用插件

#引入插件资源
source ~/.zsh/zsh-autosuggestions/zsh-autosuggestions.zsh		

#fg:字体颜色，默认为8，可设置数字0-7  bg:背景色，粗体，下划线
ZSH_AUTOSUGGEST_HIGHLIGHT_STYLE="fg=4,bg=cyan,bold,underline"

zsh-autosuggestions插件的开源地址

Homebrew

homebrew是一款Mac OS或linux系统下的软件包管理工具，通过一串简单的指令实现直接软件包安装，而不需要手动点击安装包，设置安装地址，繁琐地去安装软件。Homebrew 会将软件包安装到独立目录，并将其文件软链接至 /usr/local。

安装

/bin/bash -c "$(curl -fsSL https://raw.githubusercontent.com/Homebrew/install/master/install.sh)"

如果安装过慢或失败，请采用以下步骤：

  1. 将官网的安装文件或取下来

    curl -fsSL https://raw.githubusercontent.com/Homebrew/install/master/install >> brew_install

  2. 打开brew_install文件，切换脚本中的资源镜像链接

       vi brew_install
   
   # 替换BREW_REPO为以下地址
   BREW_REPO = “https://mirrors.ustc.edu.cn/brew.git “.freeze
   
   # 替换CORE_TAP_REPO为以下地址
CORE_TAP_REPO = “https://mirrors.ustc.edu.cn/homebrew-core.git“.freeze

  3. 运行安装脚本

    ruby brew_install

  4. 安装检测

    brew -v

资源仓库设定

到此步骤为止，您的homebrew应该是安装成功了，下面的步骤并非必要性步骤，如果您发现使用brew install [formula..]安装过慢，请继续参考该步骤。因为homebrew默认的仓库源是在github上的，访问速度较慢，所以我们切换为国内的源。

cd "$(brew --repo)"
git remote set-url origin https://mirrors.ustc.edu.cn/brew.git

cd "$(brew --repo)/Library/Taps/homebrew/"
rm -rf ./*
git clone https://mirrors.ustc.edu.cn/homebrew-core.git
git clone https://mirrors.ustc.edu.cn/homebrew-cask.git
brew update

更换homebrew-bottles镜像(影响下载速度)

# 如果您的shell使用的bash
echo 'export HOMEBREW_BOTTLE_DOMAIN=https://mirrors.tuna.tsinghua.edu.cn/homebrew-bottles' >> ~/.bash_profile
source ~/.bash_profile

#如果您的shell使用的zsh
echo 'export HOMEBREW_BOTTLE_DOMAIN=https://mirrors.tuna.tsinghua.edu.cn/homebrew-bottles' >> ~/.zshrc
source ~/.zshrc

常用命令

#brew
brew install formula...  #安装
brew search [TEXT|/REGEX/] #查找
brew list   #查看已安装的包
brew update  #更新homebrew
brew uninstall formula...  #卸载
brew info [formula...]  #查看软件包信息
brew help #查看帮助

brew cask是brew的拓展，主要负责带有GUI的软件包安装。

  1. 安装

    brew install brew-cask-completion

  2. 使用

    brew cask install    #安装软件包
...
brew cask help    #与brew命令差不多，具体请查看帮助

  3. 一键装机

      学会使用用homebrew安装软件后，我们甚至可以编写一个脚本来一键安装需要的软件，这对于一个新的系统或者系统重置之后安装软件更加便捷，只需要运行脚本，然后我们就可以离开去喝一杯咖啡，等回来后所有的软件可能已经安装成功啦。

    vi brewInstaller.sh
    #!/bin/bash
# Promgram:
#      This script installs softwares which you need
brew cask install wechat
brew install mysql
brew cask install qq
...
echo "安装已完成"
    sh brewInstaller.sh

Tree

文件树是一个特别好用的文件目录结构查看的工具，以前查看文件时不知道需要的文件在哪个目录下，一层层的cd,用了文件树后，所有的文件结构一目了然。

安装

brew install tree

显示层级目录

tree -L 1   #数字代表显示层级

支持中文

tree -N

丰富色彩

tree -C
